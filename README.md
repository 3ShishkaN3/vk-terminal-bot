# Бот VK

Данный проект — это эксперимент по созданию гибридного интерфейса управления сервером, где 95% кода написано нейросетью. Бот превращает чат ВКонтакте в полноценный терминал, позволяя выполнять прямые bash-команды или делегировать задачи ИИ-ассистенту. Главная цель — проверить, можно ли управлять через интерфейсы мессенджеров сервером, а также можно ли доверить алгоритмам системное администрирование сервера и написание кода прямо на сервере в реальном времени.
